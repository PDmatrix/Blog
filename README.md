# My personal blog

This is my attempt building application with Clean Architecture

## Architecture

### Backend
* ASP.NET Core 2.2
* CQRS
* Dapper for data access

### Identity
* ASP.NET Core 2.2
* IdentityServer4

### Frontend
* React and Next.js
* Redux
